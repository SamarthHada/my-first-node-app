# 🚀 Vercel Node.js Deployment

## 📌 Project Description
This project demonstrates the deployment of a simple Node.js application using Vercel.  
The application is implemented as a serverless function and returns a basic response.

---

## 🎯 Aim
To develop and deploy a Node.js application using Vercel cloud platform.

---

## ⚙️ Technologies Used
- Node.js
- Vercel (Serverless Platform)
- GitHub
